# `divulgacao/` — material de comunicação

Peças para **apresentar o projeto publicamente** (posts, imagens). É **apoio**, não faz parte
da metodologia (esta vive em [`recipe/`](../recipe/)) nem da pesquisa ([`lab/`](../lab/)).

| Arquivo | O que é |
|---|---|
| [`LINKEDIN-post.md`](LINKEDIN-post.md) | texto pronto para copiar + postar no LinkedIn |
| `strata-linkedin.png` | imagem (3 camadas Mneme/Morfé/Órganon + propósito) para anexar ao post |
| `strata-linkedin.svg` | fonte do gráfico (editar cores/texto e re-renderizar) |
